Summary

1. 웹 스크래핑 : 택배사 각각의 배송조회 정보구성과 데이터 형식이 다르므로, 각각의 택배사의 데이터를 따로 수집.
Request Mode에 따라 simple_html_dom을 이용하는 방법과 cURL을 이용하는 방법으로 나뉨.
2. 페이지 제작 : 운송장번호를 입력하여 스크래핑한 데이터를 제공하기 위해서 보기 편리한 UI 구조로 홈페이지 제작.
3. 프레임워크 : 정보의 보안을 위해 MVC패턴 프레임워크인 라라벨을 사용.
4. 서버 설정 : 강력한 AES-NI 하드웨어 암호화 엔진이 내장된 시놀로지(Ds918+)를 이용해 서버 설정.
5. 앱 : 사용자들이 모바일로도 배송 조회 현황을 알 수 있도록 React Native를 이용해 웹뷰 앱을 개발.
